# generateur_code_qr
Dans ce projet, je développe un générateur de QR codes en utilisant une API gratuite permettant de créer des QR codes pour n’importe quel texte ou lien. Le QR code est généré au format image, prêt à être téléchargé ou utilisé. L’application est réalisée en HTML, CSS et JavaScript, avec une interface simple et intuitive.
